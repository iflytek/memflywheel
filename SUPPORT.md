# Support

Use GitHub issues for public questions, bug reports, and feature requests:

https://github.com/iflytek/memflywheel/issues

Before filing an issue, please include:

| Field           | Example                                                                                                          |
| --------------- | ---------------------------------------------------------------------------------------------------------------- |
| Package         | `@memflywheel/core`, `@memflywheel/model`, `@memflywheel/sdk`, `@memflywheel/skills`, or `@memflywheel/adapters` |
| Runtime         | Node.js version and operating system                                                                             |
| Command         | The exact command or API call that failed                                                                        |
| Expected result | What you expected to happen                                                                                      |
| Actual result   | What happened instead                                                                                            |

Do not include API keys, tokens, private memory files, or private filesystem
paths in public issues.
