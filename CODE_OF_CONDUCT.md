# Code of Conduct

This project follows the [Iflytek Opensource Community Code of Conduct](https://github.com/iflytek/community/blob/master/code-of-conduct.md).

By participating, you are expected to uphold this code. Please report unacceptable behavior to the [Code of Conduct Committee](https://github.com/iflytek/community/blob/master/code-of-conduct/).
